# 여성신문 교육브랜드대상 브랜드 전략 AI

2026년 제21회 여성신문 학부모가 뽑은 교육브랜드대상 수상 기업 전용 AI 브랜드 전략 컨설턴트입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## Vercel 배포

1. GitHub에 이 폴더를 업로드합니다
2. vercel.com에서 GitHub 저장소를 연결합니다
3. Environment Variables에 `VITE_ANTHROPIC_API_KEY` 를 추가합니다
4. Deploy 버튼을 누릅니다

## 환경 변수

| 변수명 | 설명 |
|--------|------|
| `VITE_ANTHROPIC_API_KEY` | Anthropic API 키 (sk-ant-...) |
